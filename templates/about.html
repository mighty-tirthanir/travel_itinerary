About Itinerary Generator

About the Project

The Itinerary Generator is an AI-powered travel planner that helps users create personalized travel itineraries instantly. By leveraging machine learning and real-time travel data, it crafts optimized day-wise plans based on user preferences.

  • Personalized itinerary generation
  • User-friendly interface
  • Integrated Famous models

ML Model Used

The AI engine uses NLP and recommendation systems to create smart travel itineraries based on user input and travel data.

  • GPT-based Language Model for text generation
  • Text summarization for activity details
  • Clustering for nearby activity grouping
  • Recommendation Engine for ranking places
